# Assets Folder

## App Logo
Place your app logo here with the filename: `app_logo.png`

Recommended sizes:
- **App Icon**: 1024x1024 px (will be automatically resized for different platforms)
- **Splash Screen Logo**: 512x512 px or larger
- **In-App Logo**: 200x200 px or larger

## Usage
After adding your logo file:
1. Place the image file in this folder (e.g., `app_logo.png`)
2. Reference it in your Flutter code using: `Image.asset('assets/images/app_logo.png')`
3. Run `flutter pub get` to ensure assets are registered
